## Setup
This page makes use of the ğŸ“[News Gothic](https://www.myfonts.com/fonts/bitstream/news-gothic) font to display the score. Please make sure you have True Type copies of this font in the same directory labeled as "NEWSGOTBC_0.TFF" (bold condensed version) and "NEWSGOTC_0.TTF" (condensed version) or that the font is installed locally on your machine.

## Run
In order to make use of this application simply download index.html and open it in your preferred browser.
